Git Version Control

Git is a distributed version control system that tracks changes in source code during software development. It enables multiple developers to work together on the same codebase efficiently.

Git Workflow

Key Concepts:

  • Repository: Storage location for your project
  • Commit: Snapshot of changes
  • Branch: Independent line of development
  • Merge: Combining changes from different branches
  • Remote: Hosted version of your repository
